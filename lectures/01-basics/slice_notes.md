# Slice Notes

## Idiomatic slicing patterns

```
>>> s = 'Slicing example!'
>>> s[4:]                  # Drop the first 4 characters
'ing example!'
>>> s[-4:]                 # Take the last 4 characters
'ple!'
>>> s[:4]                  # Take the first 4 characters
'Slic'
>>> s[:-4]                 # Drop the last 4 characters
'Slicing exam'
```
